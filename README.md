PROJECT OVERVIEW

This project focuses on Automated Report Generation Using Python, where student performance data is read from a CSV file, analyzed programmatically, and converted into a well-formatted PDF report. The main objective of this code is to eliminate manual report preparation by automating data analysis and document creation.

The script uses Python as the core programming language and integrates data processing and document generation libraries to achieve its goal. The input data is stored in a CSV file (data.csv) that contains student names and their corresponding marks. The program reads this file, performs basic statistical analysis such as calculating the average, highest, and lowest marks, and then generates a professional-looking PDF report summarizing both the raw data and the analysis results.

The generated PDF, named Automated_Report.pdf, includes a title, subtitle, a list of student marks, and an analysis summary section. This makes the report suitable for academic use, institutional documentation, and automated reporting systems.

Overall, the code demonstrates how data analytics and document automation can be combined using Python to create efficient and reusable reporting solutions.

PLOT FORM USED

No plot or graphical chart is used in this project.

Instead of graphical visualization, the project uses a text-based PDF report format. The focus of this code is on structured document generation, not on plotting or chart visualization. All information, including student marks and statistical analysis, is presented in a formatted textual layout within the PDF.

The absence of plots makes this project ideal for scenarios where formal documentation is required, such as examination reports, academic records, or administrative summaries. However, graphical plots (such as bar charts or pie charts) can be added in future enhancements if visual analysis is needed.
